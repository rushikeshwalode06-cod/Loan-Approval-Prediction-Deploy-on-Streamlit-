# Loan Approval Prediction Deploy on Streamlit Using Logistic Regression 



## 📌 Project Overview

This project is a **Machine Learning-based Loan Approval Prediction System** developed using **Logistic Regression**. The application predicts whether a loan application is likely to be **Approved** or **Rejected** based on applicant details entered by the user.

The trained model has been deployed using **Streamlit**, providing an interactive and user-friendly web interface for real-time predictions.

---

## 🚀 Features

* Predicts loan approval status in real time
* Clean and interactive Streamlit interface
* Logistic Regression-based classification model
* Fast and accurate predictions
* Simple input form for users

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Logistic Regression
* Streamlit
* Joblib

---

## 📊 Input Features

The model accepts the following inputs:

* Gender
* Married
* Applicant Income
* Loan Amount
* Credit History

---

## 🎯 Output

The model predicts one of the following outcomes:

* ✅ Loan Approved
* ❌ Loan Rejected

---

## 🧠 Machine Learning Model

**Algorithm Used:** Logistic Regression

Logistic Regression is a supervised machine learning algorithm widely used for binary classification problems. In this project, it classifies loan applications into two categories:

* Loan Approved ✅
* Loan Rejected ❌

## 📈 Workflow

1. Load the trained Logistic Regression model.
2. User enters applicant details.
3. Input data is processed.
4. The model predicts the loan approval status.
5. The prediction is displayed instantly.
